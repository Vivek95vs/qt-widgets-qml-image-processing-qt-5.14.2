# qt-widgets-qml-image-processing-qt-5.14.2
This repository contains multiple Qt-based C++ projects focused on image processing and visualization. It includes QtWidgets implementations developed using Qt 5.14.2 (MSVC 2017), ITK 5.2(Insight Segmentation and Registration Toolkit),VTK 8.0\9.2.6 (Visualization Toolkit),DCMTK 3.6.6(DICOM Toolkit) and OpenCV 4.6.0.
